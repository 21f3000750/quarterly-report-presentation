Financial Modeling: CAGR

We use the Compound Annual Growth Rate (CAGR) to measure long-term growth.

The formula is:

$$ \text{CAGR} = \left( \frac{\text{Ending Value}}{\text{Beginning Value}} \right)^{\frac{1}{n}} - 1 $$

Where: $n$ is the number of years.

Here you can see the standard formula for CAGR. This helps us smooth out volatility and understand the true growth trend over time. For our main product, the 5-year CAGR is an impressive 18%.

Code Example: Risk Calculation

A simplified Python function to calculate Value at Risk (VaR).


import numpy as np

def calculate_var(returns, confidence_level=0.95):
  """Calculates the Value at Risk (VaR) for a portfolio."""
  sorted_returns = np.sort(returns)
  index = int((1 - confidence_level) * len(sorted_returns))
  return sorted_returns[index]
Our risk management models are, of course, far more complex. However, this Python snippet illustrates the core statistical concept behind VaR calculation, which is a cornerstone of our market risk assessment.
